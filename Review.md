# Review Questions

## What is Node.js?

Node.js is a server environment that utilizes javascript.

## What is Express?

Express is a node web applciation framework that provides users with a way to very rapidly ddevelop a node based application.

## Mention two parts of Express that you learned about this week.

express can handles request and response functionalities such as res.json() and req.params

## What is Middleware?

Middleware are functions that occur in a sequential order through the use of the next() function. Each middleware function can provide unique functionalities such as logging information.

## What is a Resource?

A resource is everything that can be accessed in a database through a specific url or endpoint.

## What can the API return to help clients know if a request was successful?

An API can return an HTTP status code.

## How can we partition our application into sub-applications?

This can be done with Express router.

## What is express.json() and why do we need it?

express.json() ensures that data that is being passed through is in a json format.
